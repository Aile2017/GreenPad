## 外部フィルタ機能

### 概要

編集中のテキストを外部プログラムに渡し、その出力で置き換える機能。

### 動作仕様

- 選択範囲がある場合: 選択部分のみを外部プログラムの stdin に渡し、stdout の内容で選択部分を置換する
- 選択範囲がない場合: ファイル全体を stdin に渡し、stdout の内容でファイル全体を置換する

### 起動方法

- メニュー: Edit > Modify サブメニューに追加
- ショートカットキー: `Ctrl+\`
- ショートカットキーでコマンドライン入力ダイアログを表示し、入力されたコマンドを `cmd.exe /c` 経由で実行する

### エンコーディング / 改行コード

- 編集中のファイルのエンコーディング・改行コードのままバイト列として stdin に渡す
  - 「そのファイルを保存して外部プログラムに渡した場合」と同じ状況になるイメージ
- stdout の内容も同じエンコーディングとして解釈して置換する

### エラー処理

- フィルタプログラムの終了コードが非ゼロの場合は置換を行わず、元のテキストを保持する
- 終了コードが非ゼロの場合はエラーダイアログを表示する
  - stderr に出力があればその内容をダイアログに含めて表示する
  - stderr が空の場合は「終了コード: N」のみ表示する

### `cmd.exe /c` 経由で実行する理由

- `sort`, `find` 等の組み込みコマンドが使える
- `cmd1 | cmd2` のようなパイプチェーンが書ける
- `2>&1 |` で stderr を他のプログラムに渡すことができる
- `cmd1 && cmd2` 等の条件実行も使える

### コマンド入力ダイアログ

- ウィンドウタイトル: `External Filter` / `外部フィルタ`
- コンボボックス上のラベル: `Command:` / `コマンド:`
- ボタン: `Run` / `実行` と `Cancel` / `キャンセル`

```
┌─ External Filter ──────────────────────┐
│ Command:                               │
│ [ sort -r                         ]▼  │
│                                        │
│                    [  Run  ] [Cancel]  │
└────────────────────────────────────────┘
```

- 入力エリアはコンボボックス
  - ↑↓キーで履歴を入力欄に表示
  - ▼ボタンでドロップダウンリストを開く
  - 既存の検索ダイアログのコンボボックスと同じ操作感
- 履歴は ini ファイルで管理 (最近使ったファイルと同様の仕組み)
  - 最大10件を保持 (将来的には設定で変更できるようにしたい)
  - `CreateProcess` の起動に成功したコマンドを履歴に追加する
    - 起動失敗 (コマンドが見つからない等) は追加しない
    - プロセスが起動したが終了コード非ゼロの場合は追加する

### 実装メモ (未着手)

- ダイアログは既存の `DlgImpl` を使った一行入力ダイアログ
- `CreateProcess` + 匿名パイプで stdin/stdout/stderr を接続する
- フィルタ実行中は UI をブロックする (編集内容が壊れるのを防ぐため、制御を返さない)
- パイプバッファのデッドロック回避のためスレッドを使う
  - stdin への書き込みと stdout/stderr の読み出しを並走させる
  - 構成例:
    - 書き込みスレッド : stdin に全データを WriteFile で送り込み、完了後 CloseHandle で EOF を通知
    - メインスレッド   : stdout を ReadFile で読み続ける
    - stderr スレッド  : stderr を ReadFile で読み続ける (書き込みスレッドと兼務でも可)
  - 全スレッド完了後、`WaitForSingleObject` で子プロセスの終了を待つ
  - 終了コードを取得して成否を判定する
- kilib/thread.cpp の既存スレッドラッパーを活用する
- ダイアログのコントロールサイズはロシア語に合わせる
  - `[Run]` ボタンは `Выполнить` (10文字) に合わせた幅を確保する
  - ダイアログ幅・コンボボックス幅もキャプション `Внешний фильтр` を基準にする
  - エラーメッセージは `MessageBox` の自動折り返しに任せるため特別な対応不要
